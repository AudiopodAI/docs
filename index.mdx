### Quick SDK Setup

<Tabs>
  <Tab title="Python">
    ```python
    # Install and setup
    pip install audiopod
    
    from audiopod import Client
    client = Client()  # Uses AUDIOPOD_API_KEY env var
    
    # Generate speech
    speech = client.text_to_speech.create(
        text="Hello, world!",
        voice_id="aura"
    )
    ```
  </Tab>
  <Tab title="Node.js">
    ```javascript
    // Install and setup
    npm install audiopod-js
    
    const { AudioPodClient } = require('audiopod-js');
    const client = new AudioPodClient(); // Uses AUDIOPOD_API_KEY env var
    
    // Generate speech
    const speech = await client.textToSpeech.create({
      text: "Hello, world!",
      voiceId: "aura"
    });
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    # Set your API key
    export AUDIOPOD_API_KEY="your_api_key_here"
    
    # Generate speech
    curl -X POST "https://api.audiopod.ai/api/v1/voice/voices/aura/generate" \
      -H "Authorization: Bearer $AUDIOPOD_API_KEY" \
      -F "input_text=Hello, world!" \
      -F "audio_format=mp3"
    ```
  </Tab>
</Tabs>
