## Overview

The AudioPod AI Noise Reduction API provides state-of-the-art audio denoising capabilities using deep learning models and spectral gating techniques. Our advanced system removes background noise, enhances speech clarity, and improves overall audio quality for podcasts, recordings, and real-time communication applications.

<Info>
  Our noise reduction system combines multiple approaches including deep learning neural networks, advanced spectral gating algorithms, and adaptive filtering to achieve superior results across different noise types and audio conditions. The system automatically selects the best approach based on audio characteristics.
</Info>

### Real-time Denoising

Process audio streams in real-time for live applications.

<Tabs>
  <Tab title="WebSocket">
    ```javascript
    // WebSocket real-time denoising
    const ws = new WebSocket('wss://api.audiopod.ai/ws/v1/denoiser');
    
    ws.onopen = () => {
      // Authenticate
      ws.send(JSON.stringify({
        type: 'auth',
        token: 'your_access_token',
        config: {
          method: 'spectral_gating',
          strength: 0.7,
          latency_mode: 'low'
        }
      }));
    };
    
    ws.onmessage = (event) => {
      const data = JSON.parse(event.data);
      
      if (data.type === 'audio_processed') {
        // Received denoised audio chunk
        const cleanAudio = data.audio_data;
        playAudioChunk(cleanAudio);
      }
    };
    
    // Send audio chunks for processing
    function sendAudioChunk(audioBuffer) {
      ws.send(JSON.stringify({
        type: 'process_chunk',
        audio_data: arrayBufferToBase64(audioBuffer)
      }));
    }
    ```
  </Tab>
  <Tab title="Python">
    ```python
    import asyncio
    import websockets
    import json
    import base64
    
    async def real_time_denoise():
        uri = "wss://api.audiopod.ai/ws/v1/denoiser"
        
        async with websockets.connect(uri) as websocket:
            # Authenticate
            auth_message = {
                "type": "auth",
                "token": access_token,
                "config": {
                    "method": "spectral_gating",
                    "strength": 0.7,
                    "latency_mode": "low"
                }
            }
            await websocket.send(json.dumps(auth_message))
            
            # Process audio chunks
            async for message in websocket:
                data = json.loads(message)
                
                if data["type"] == "audio_processed":
                    clean_audio = base64.b64decode(data["audio_data"])
                    # Handle denoised audio
                    process_clean_audio(clean_audio)
    
    asyncio.run(real_time_denoise())
    ```

## Denoising Methods

### Deep Learning Model

Our primary denoising method using advanced neural networks.

**Features:**

- Superior noise reduction quality
- Preserves speech naturalness
- Handles complex noise patterns
- Adaptive to different audio types

```json
  "strength": 0.8,
  "preserve_speech": true,
  "adaptive_threshold": true
```

### Spectral Gating

Traditional but effective spectral subtraction method.

**Features:**

- Fast processing
- Low computational requirements
- Good for stationary noise
- Real-time capable

**Configuration:**

```json
{
  "method": "spectral_gating",
  "gate_threshold": -20,
  "frequency_smoothing": 0.05,
  "time_smoothing": 0.1
}
```

Combines multiple denoising techniques for optimal results.

**Configuration:**


Monitor processing progress for long-running jobs:

    ```http

    Authorization: Bearer {access_token}
    ```
  </Tab>
  <Tab title="Python">
    ```python
    def wait_for_completion(job_id, timeout=300):
        start_time = time.time()
        
        while time.time() - start_time < timeout:
            response = requests.get(
                f"https://api.audiopod.ai/api/v1/denoiser/jobs/{job_id}",
                headers={"Authorization": f"Bearer {access_token}"}
            )
            
            job = response.json()
            status = job["status"]
            
            if status == "completed":
                return job["result"]
            elif status == "failed":
                raise Exception(f"Job failed: {job['error']}")
            
            print(f"Status: {status} - Progress: {job.get('progress', 0)}%")
            time.sleep(5)
        
        raise TimeoutError("Job did not complete within timeout")
    
    result = wait_for_completion(job_id)
    ```

### Processing Parameters

| ------------------ | ------- | ------------------------------ | ----------- |

## Best Practices

### Optimization Tips

1. **Choose the right method**: Deep learning for quality, spectral gating for speed
2. **Use appropriate noise profiles**: Improves results significantly
3. **Adjust strength carefully**: Too high may degrade speech quality
4. **Preprocess when possible**: Remove obvious artifacts before denoising
5. **Monitor quality metrics**: Use feedback to tune parameters
6. **Batch similar content**: Group files with similar characteristics

### Quality Guidelines

1. **Input Requirements**:

   - Minimum sample rate: 16 kHz
   - Recommended: 44.1 kHz or higher
   - Avoid heavily compressed audio
   - Ensure sufficient audio levels

2. **Parameter Selection**:
   - Start with strength 0.7-0.8
   - Use preserve_speech for voice content
   - Custom profiles for consistent environments
   - Real-time mode for latency-critical applications

### Integration Examples

```python
# Production pipeline example
class AudioProcessor:
    def __init__(self, api_key):
        self.client = AudioPodClient(api_key)
        self.quality_threshold = 0.7

    def process_audio(self, file_path, output_path):
        # Analyze quality first
        quality = self.client.audio.analyze_quality(file_path)

        if quality.noise_level > self.quality_threshold:
            # Apply denoising
            result = self.client.denoiser.process(
                file_path=file_path,
                method="deep_learning",
                strength=min(quality.noise_level, 0.9)
            )

            # Download processed audio
            self.client.download_file(result.audio_url, output_path)
            return result.metrics
        else:
            # Audio quality sufficient, no processing needed
            shutil.copy(file_path, output_path)
            return quality
```
