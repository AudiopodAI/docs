## Overview

The AudioPod AI Account Management API provides complete control over user accounts, including profile management, usage tracking, subscription handling, and administrative functions. Manage user data, monitor API usage, and configure account settings programmatically.

<Info>
  Account management endpoints require proper authentication and may have
  different permission levels. Some operations require administrative privileges
  or specific API key scopes.
</Info>

    ```http
    GET /account/profile
    Authorization: Bearer {access_token}
    ```

### Usage by Date Range

Get usage analytics for specific time periods:

<Tabs>
  <Tab title="GET">
    ```http
    GET /account/usage/range?start_date=2024-01-01&end_date=2024-01-15&granularity=daily
    Authorization: Bearer {access_token}
    ```
  </Tab>
  <Tab title="Python">
    ```python
    from datetime import datetime, timedelta
    
    end_date = datetime.now()
    start_date = end_date - timedelta(days=30)
    
    response = requests.get(
        "https://api.audiopod.ai/api/v1/account/usage/range",
        headers={"Authorization": f"Bearer {access_token}"},
        params={
            "start_date": start_date.isoformat(),
            "end_date": end_date.isoformat(),
            "granularity": "daily",
            "group_by": "service"
        }
    )
    
    usage_range = response.json()
    
    # Plot usage over time
    import matplotlib.pyplot as plt
    
    dates = [item['date'] for item in usage_range['data']]
    credits = [item['credits'] for item in usage_range['data']]
    
    plt.plot(dates, credits)
    plt.title('Daily Credit Usage')
    plt.xlabel('Date')
    plt.ylabel('Credits Used')
    plt.show()
    ```
  </Tab>
</Tabs>

### Export Usage Data

Export detailed usage data for analysis:

<Tabs>
  <Tab title="POST">
    ```http
    POST /account/usage/export
    Authorization: Bearer {access_token}
    Content-Type: application/json
    
    {
      "format": "csv",
      "date_range": {
        "start": "2024-01-01",
        "end": "2024-01-31"
      },
      "include": [
        "requests",
        "credits",
        "errors",
        "processing_times"
      ],
      "group_by": "service"
    }
    ```
  </Tab>
  <Tab title="Python">
    ```python
    response = requests.post(
        "https://api.audiopod.ai/api/v1/account/usage/export",
        headers={"Authorization": f"Bearer {access_token}"},
        json={
            "format": "csv",
            "date_range": {
                "start": "2024-01-01",
                "end": "2024-01-31"
            },
            "include": [
                "requests",
                "credits", 
                "errors",
                "processing_times"
            ],
            "group_by": "service"
        }
    )
    
    export_job = response.json()
    job_id = export_job["job_id"]
    
    # Poll for completion
    while True:
        status_response = requests.get(
            f"https://api.audiopod.ai/api/v1/account/usage/export/{job_id}",
            headers={"Authorization": f"Bearer {access_token}"}
        )
        
        status = status_response.json()
        if status["status"] == "completed":
            download_url = status["download_url"]
            print(f"Export ready: {download_url}")
            break
        
        time.sleep(2)
    ```
  </Tab>
</Tabs>

### Delete Account

Permanently delete account and all associated data:

<Tabs>
  <Tab title="DELETE">
    ```http
    DELETE /account
    Authorization: Bearer {access_token}
    Content-Type: application/json
    
    {
      "confirmation": "DELETE",
      "reason": "no_longer_needed",
      "feedback": "Found a different solution"
    }
    ```
  </Tab>
  <Tab title="Python">
    ```python
    # Account deletion requires explicit confirmation
    response = requests.delete(
        "https://api.audiopod.ai/api/v1/account",
        headers={"Authorization": f"Bearer {access_token}"},
        json={
            "confirmation": "DELETE",
            "reason": "no_longer_needed",
            "feedback": "Found a different solution"
        }
    )
    ```

## Best Practices

```python
# Production account management class
class AccountManager:
    def __init__(self, api_key):
        self.client = AudioPodClient(api_key)
        self.notification_handlers = {}

    def setup_monitoring(self):
        # Set up credit monitoring
        profile = self.client.account.get_profile()
        credit_threshold = profile.credits.monthly_allowance * 0.1

        self.client.account.update_notifications({
            "thresholds": {
                "low_credit_warning": credit_threshold
            },
            "webhook_notifications": {
                "enabled": True,
                "url": "https://myapp.com/webhooks/credits",
                "events": ["credits.low"]
            }
        })

    def get_usage_summary(self, days=30):
        usage = self.client.account.get_usage(period=f"{days}d")

        return {
            "total_requests": usage.summary.total_requests,
            "credits_used": usage.summary.credits_used,
            "success_rate": usage.summary.success_rate,
            "top_service": max(usage.breakdown, key=lambda x: x.requests).service,
            "daily_average": usage.summary.credits_used / days
        }

    def optimize_api_keys(self):
        keys = self.client.account.list_api_keys()

        for key in keys:
            if not key.last_used_at:
                # Unused key
                print(f"Consider removing unused key: {key.name}")
            elif self._is_old_key(key):
                # Old key
                print(f"Consider rotating old key: {key.name}")

    def _is_old_key(self, key):
        from datetime import datetime, timedelta
        created = datetime.fromisoformat(key.created_at.replace('Z', '+00:00'))
        return datetime.now() - created > timedelta(days=365)
```
