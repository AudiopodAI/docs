## Track Interactions

### Like a Track

#### `POST /api/v1/music/jobs/{job_id}/like`

Like a music track. Requires authentication.

```bash
curl -X POST "https://api.audiopod.ai/api/v1/music/jobs/123/like" \
  -H "X-API-Key: $AUDIOPOD_API_KEY"
```

#### `DELETE /api/v1/music/jobs/{job_id}/like`

Remove a like from a track (toggle).

```bash
curl -X DELETE "https://api.audiopod.ai/api/v1/music/jobs/123/like" \
  -H "X-API-Key: $AUDIOPOD_API_KEY"
```

### Dislike a Track

#### `POST /api/v1/music/jobs/{job_id}/dislike`

Dislike a music track. Requires authentication.

```bash
curl -X POST "https://api.audiopod.ai/api/v1/music/jobs/123/dislike" \
  -H "X-API-Key: $AUDIOPOD_API_KEY"
```

#### `DELETE /api/v1/music/jobs/{job_id}/dislike`

Remove a dislike from a track (toggle).

```bash
curl -X DELETE "https://api.audiopod.ai/api/v1/music/jobs/123/dislike" \
  -H "X-API-Key: $AUDIOPOD_API_KEY"
```

**Response (all interaction endpoints):**

```json
{
  "success": true,
  "interaction_type": "like",
  "like_count": 5,
  "dislike_count": 1
}
```

## Share a Track

### `POST /api/v1/music/jobs/{job_id}/share`

Share a music track and generate a shareable URL. Works for both authenticated and anonymous users.

```bash
curl -X POST "https://api.audiopod.ai/api/v1/music/jobs/123/share" \
  -H "X-API-Key: $AUDIOPOD_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"platform": "twitter"}'
```

**Request Body:**

| Parameter | Type | Description |
|-----------|------|-------------|
| `platform` | string | Sharing platform (e.g. `twitter`, `facebook`, `link`) |
| `additional_data` | object | Optional extra metadata |

**Response:**

```json
{
  "share_token": "abc123xyz",
  "share_url": "https://audiopod.ai/shared/abc123xyz",
  "platform": "twitter"
}
```

## Comments

### Add Comment

#### `POST /api/v1/music/jobs/{job_id}/comments`

Add a comment to a track. Requires authentication.

```bash
curl -X POST "https://api.audiopod.ai/api/v1/music/jobs/123/comments" \
  -H "X-API-Key: $AUDIOPOD_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"content": "Great track! Love the melody."}'
```

### List Comments

#### `GET /api/v1/music/jobs/{job_id}/comments`

Get comments for a track.

```bash
curl "https://api.audiopod.ai/api/v1/music/jobs/123/comments?limit=20" \
  -H "X-API-Key: $AUDIOPOD_API_KEY"
```

**Query Parameters:** `skip` (int), `limit` (int, default 50)

## Shared Tracks (Public Access)

These endpoints provide public access to shared tracks. No authentication required.

### Get Shared Track

#### `GET /api/v1/music/share/{share_token}`

Get shared track details by share token.

```bash
curl "https://api.audiopod.ai/api/v1/music/share/abc123xyz"
```

#### `GET /api/v1/music/shared/{share_token}`

Alternative endpoint for shared track details.

```bash
curl "https://api.audiopod.ai/api/v1/music/shared/abc123xyz"
```

**Response:** Full track data including audio URLs, display name, lyrics, LRC content, and interaction stats.

### Download Shared Track

#### `GET /api/v1/music/shared/{share_token}/download`

Get a download URL for a shared track in a specific format.

```bash
# Download as WAV
curl "https://api.audiopod.ai/api/v1/music/shared/abc123xyz/download?format=wav"

# Download as MP3
curl "https://api.audiopod.ai/api/v1/music/shared/abc123xyz/download?format=mp3"
```

**Query Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `format` | string | `"wav"` | Download format: `wav`, `mp3`, `flac` |

### Shared Track Stats

#### `GET /api/v1/music/share/{share_token}/stats`

Get public statistics for a shared track.

```bash
curl "https://api.audiopod.ai/api/v1/music/share/abc123xyz/stats"
```

### Like/Dislike Shared Track

#### `POST /api/v1/music/shared/{share_token}/like`

Like a shared track. Works for both authenticated and anonymous users.

```bash
curl -X POST "https://api.audiopod.ai/api/v1/music/shared/abc123xyz/like"
```

#### `POST /api/v1/music/shared/{share_token}/dislike`

Dislike a shared track.

```bash
curl -X POST "https://api.audiopod.ai/api/v1/music/shared/abc123xyz/dislike"
```

### Comments on Shared Tracks

#### `POST /api/v1/music/shared/{share_token}/comments`

Add a comment to a shared track.

```bash
curl -X POST "https://api.audiopod.ai/api/v1/music/shared/abc123xyz/comments" \
  -H "Content-Type: application/json" \
  -d '{"content": "Amazing song!", "author_name": "Anonymous"}'
```

#### `GET /api/v1/music/shared/{share_token}/comments`

List comments on a shared track.

```bash
curl "https://api.audiopod.ai/api/v1/music/shared/abc123xyz/comments?limit=20"
```

## Featured Tracks (Music Radio)

Public endpoints for browsing and discovering featured music. No authentication required.

### List Featured Tracks

#### `GET /api/v1/music/featured`

Get featured/showcase tracks with filtering and sorting.

```bash
# Get top quality featured tracks
curl "https://api.audiopod.ai/api/v1/music/featured?sort_by=quality_score&page=1&page_size=20"

# Filter by language and genre
curl "https://api.audiopod.ai/api/v1/music/featured?language=en&genre=pop"
```

**Query Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `language` | string | null | Filter by vocal language |
| `genre` | string | null | Filter by genre |
| `emotion` | string | null | Filter by emotion |
| `occasion` | string | null | Filter by occasion |
| `task_type` | string | null | Filter by task type |
| `sort_by` | string | `"quality_score"` | Sort: `quality_score`, `created_at`, `random` |
| `page` | int | 1 | Page number |
| `page_size` | int | 20 | Items per page (max 100) |

### Random Tracks (Radio Shuffle)

#### `GET /api/v1/music/radio/random`

Get random featured tracks for Music Radio shuffle mode.

```bash
curl "https://api.audiopod.ai/api/v1/music/radio/random?count=10&language=en"
```

**Query Parameters:**

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `count` | int | 10 | Number of tracks (max 50) |
| `exclude_ids` | string | null | Comma-separated IDs to exclude (already played) |
| `language` | string | null | Filter by language |
| `genre` | string | null | Filter by genre |

**Rate limit:** 120 requests per minute.

### Radio Generate

#### `POST /api/v1/music/radio/generate`

Generate music on-the-fly for Music Radio using turbo mode (8-step inference for fast generation).

```bash
curl -X POST "https://api.audiopod.ai/api/v1/music/radio/generate" \
  -H "Content-Type: application/json" \
  -d '{"query": "upbeat dance music"}'
```

**Note:** Radio generation is free (no credits charged). Downloads are gated behind authentication and credits.

**Rate limit:** 3 requests per minute.

## Public Demo

Rate-limited endpoints for trying music generation without an account.

### Get Demo Limits

#### `GET /api/v1/music/public/limits`

Get public demo usage limits and current status.

```bash
curl "https://api.audiopod.ai/api/v1/music/public/limits"
```

**Response:**

```json
{
  "daily_limit": 3,
  "used": 1,
  "remaining": 2,
  "max_duration_seconds": 30,
  "available_tasks": ["text2music", "text2rap", "prompt2instrumental"]
}
```

### Generate (Demo)

#### `POST /api/v1/music/public/generate`

Generate music for demo — preview only, no download. Limited to 3 uses per day per IP, 30 seconds max.

```bash
curl -X POST "https://api.audiopod.ai/api/v1/music/public/generate" \
  -H "Content-Type: multipart/form-data" \
  -F "prompt=upbeat pop song with catchy melody" \
  -F "task=text2music" \
  -F "duration=30"
```

Returns streaming audio response (`audio/mpeg`).

### Generate Shareable (Demo)

#### `POST /api/v1/music/public/generate-shareable`

Generate music for demo with a permanent shareable link for social sharing.

```bash
curl -X POST "https://api.audiopod.ai/api/v1/music/public/generate-shareable" \
  -H "Content-Type: multipart/form-data" \
  -F "prompt=energetic dance track" \
  -F "task=text2music" \
  -F "title=My Demo Song"
```

**Response:**

```json
{
  "success": true,
  "audio_url": "https://media.audiopod.ai/...",
  "share_url": "https://audiopod.ai/shared/abc123",
  "share_token": "abc123",
  "job_id": 789,
  "display_name": "My Demo Song",
  "duration": 30.0,
  "task": "text2music"
}
```

**Rate limit:** 5 requests per minute.
