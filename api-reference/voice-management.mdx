## Overview

The Voice Management API allows you to discover available voices, create and manage custom voices, and organize voice collections. This API is essential for applications that need dynamic voice selection and custom voice creation capabilities.

### Base URL

```
https://api.audiopod.ai/api/v1
```

### Authentication

All voice management endpoints require authentication using your API key in the Authorization header:

```
Authorization: Bearer your_api_key_here
```

## Available Voices

### List Public Voices

Get all publicly available pre-built voices.

<Tabs>
  <Tab title="GET">
    ```http
    GET /voices/public
    ```
  </Tab>
  <Tab title="Python">
    ```python
    response = client.voices.list_public(
        language="en",           # Optional: filter by language
        gender="female",         # Optional: filter by gender

        limit=50,               # Optional: limit results
    ```

### Create Custom Voice

Create a new custom voice by training on audio samples.

<Tabs>
  <Tab title="POST">
    ```http
    POST /voices/custom
    Content-Type: multipart/form-data
    ```
  </Tab>
  <Tab title="Python">
    ```python
    # Single audio sample
    with open("voice_sample.mp3", "rb") as audio_file:
        response = client.voices.create_custom(
            name="Sarah's Narrator Voice",
            description="Professional female voice for audiobook narration",
            sample_audio=audio_file,
            language="en",
            gender="female",
            use_case="narration",
            training_options={
                "quality": "high",
                "multilingual": True,
                "preserve_style": True
            }
        )
    
    print(f"Voice ID: {response.id}")
    print(f"Status: {response.status}")
    print(f"Training started: {response.training_started_at}")
    
    # Multiple audio samples for better quality
    sample_files = [
        "sample1_reading.mp3",
        "sample2_conversation.mp3", 
        "sample3_presentation.mp3"
    ]
    
    response = client.voices.create_custom(
        name="High Quality Custom Voice",
        description="Multi-sample trained voice for premium quality",
        sample_files=sample_files,
        language="en",
        training_options={
            "quality": "premium",
            "multilingual": True,
            "training_duration": "extended"
        }
    )
    ```
  </Tab>
  <Tab title="Node.js">
    ```javascript
    const fs = require('fs');
    
    // Single sample
    const audioSample = fs.readFileSync('voice_sample.mp3');
    
    const response = await client.voices.createCustom({
      name: "Sarah's Narrator Voice",
      description: "Professional female voice for audiobook narration",
      sampleAudio: audioSample,
      language: 'en',
      gender: 'female',
      useCase: 'narration',
      trainingOptions: {
        quality: 'high',
        multilingual: true,
        preserveStyle: true
      }
    });
    
    console.log(`Voice ID: ${response.id}`);
    console.log(`Status: ${response.status}`);
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    curl -X POST "https://api.audiopod.ai/api/v1/voice/voices/" \
      -H "Authorization: Bearer your_api_key" \
      -F "name=Sarah's Narrator Voice" \
      -F "description=Professional female voice for audiobook narration" \
      -F "sample_audio=@voice_sample.mp3" \
      -F "language=en" \
      -F "gender=female" \
      -F "use_case=narration" \
      -F "training_options[quality]=high" \
      -F "training_options[multilingual]=true"
    ```
  </Tab>
</Tabs>

#### Request Parameters

| Parameter          | Type   | Required | Description                                                     |
| ------------------ | ------ | -------- | --------------------------------------------------------------- |
| `name`             | string | Yes      | Display name for the custom voice                               |
| `description`      | string | No       | Description of the voice and its intended use                   |
| `sample_audio`     | file   | Yes\*    | Primary audio sample (MP3, WAV, M4A)                            |
| `sample_files`     | file[] | Yes\*    | Multiple audio samples for higher quality                       |
| `language`         | string | Yes      | Primary language code (e.g., "en", "es")                        |
| `gender`           | string | No       | Voice gender ("male", "female", "neutral")                      |
| `age_range`        | string | No       | Age range ("young", "adult", "mature")                          |
| `use_case`         | string | No       | Intended use case ("narration", "conversation", "presentation") |
| `training_options` | object | No       | Training configuration options                                  |

\*Either `sample_audio` or `sample_files` is required

#### Training Options

| Option              | Type    | Description                                                 |
| ------------------- | ------- | ----------------------------------------------------------- |
| `quality`           | string  | Training quality ("standard", "high", "premium")            |
| `multilingual`      | boolean | Enable multilingual capabilities                            |
| `preserve_style`    | boolean | Preserve original speaking style                            |
| `training_duration` | string  | Training time ("fast", "standard", "extended")              |
| `emotion_range`     | string  | Emotional range ("limited", "normal", "extended")           |
| `noise_tolerance`   | string  | Background noise handling ("strict", "moderate", "lenient") |

### Advanced Voice Search

  <Tab title="POST">
    ```http
    POST /voices/search
    Content-Type: application/json
    ```
  </Tab>
  <Tab title="Python">
    ```python
    results = client.voices.search(
        query="professional female narrator",
        filters={
            "languages": ["en", "es"],
            "gender": "female",
            "use_cases": ["narration", "presentation"],
            "characteristics": {
                "clarity": {"min": 0.8},
                "warmth": {"min": 0.6},
                "energy_level": ["medium", "high"]
            }
        },
        include_custom=True,
        sort_by="popularity",
        limit=20
    )
    
    for voice in results.voices:
        print(f"{voice.name}: {voice.match_score}")
    ```
  </Tab>
  <Tab title="Node.js">
    ```javascript
    const results = await client.voices.search({
      query: 'professional female narrator',
      filters: {
        languages: ['en', 'es'],

        characteristics: {
          clarity: { min: 0.8 },
          warmth: { min: 0.6 },
          energyLevel: ['medium', 'high']
        }
    ```

## Best Practices

### Voice Creation Tips

**✅ Recommended:**

- Use high-quality audio samples (16kHz+, minimal noise)
- Provide 30 seconds to 5 minutes of sample audio
- Use natural, conversational speech
- Include variety in tone and pacing
- Test with sample text before full deployment

**❌ Avoid:**

- Low-quality or compressed audio
- Background music or noise
- Robotic or synthetic-sounding samples
- Extremely short samples (< 10 seconds)
- Multiple speakers in the same sample

```python
def discover_perfect_voice(requirements):
    """Find the best voice based on specific requirements"""

    # Search for voices matching requirements
    search_results = client.voices.search(
        query=requirements["description"],
        filters={
            "languages": requirements["languages"],
            "gender": requirements.get("gender"),
            "use_cases": requirements["use_cases"],
            "characteristics": requirements.get("characteristics", {})
        },
        include_custom=True,
        sort_by="match_score",
        limit=10
    )

    # Test top candidates
    top_voices = search_results.voices[:3]
    test_results = []

        # Generate test audio

            voice_id=voice.id
        )

        test_results.append({
            "voice": voice,
            "test_audio": test_audio.audio_data,
            "match_score": voice.match_score
        })

    return test_results
```

### Custom Voice Training Pipeline

```python
def create_and_monitor_custom_voice(voice_config):
    """Complete pipeline for custom voice creation"""

    # Validate audio samples
    for sample_file in voice_config["sample_files"]:
        validation = client.voices.validate_sample(sample_file)
        if not validation.is_valid:
            raise ValueError(f"Invalid sample {sample_file}: {validation.issues}")

    # Create custom voice
    voice_creation = client.voices.create_custom(
        name=voice_config["name"],
        description=voice_config["description"],
        sample_files=voice_config["sample_files"],
        language=voice_config["language"],
        training_options=voice_config.get("training_options", {})
    )

    print(f"Voice training started: {voice_creation.id}")

    # Monitor training progress
    while True:
        status = client.voices.get_training_status(voice_creation.id)

        print(f"Training progress: {status.training_progress}%")

        if status.status == "ready":
            print(f"Voice training completed! Quality score: {status.quality_score}")

            # Test the new voice
            test_audio = client.text_to_speech.create(
                text="This is a test of my new custom voice.",
                voice_id=voice_creation.id
            )

            return {
                "voice_id": voice_creation.id,
                "quality_score": status.quality_score,
                "test_audio": test_audio.audio_data,
                "sample_url": status.sample_audio_url
            }

        elif status.status == "failed":
            raise Exception(f"Voice training failed: {status.error_message}")

        # Wait before checking again
        time.sleep(30)

# Example usage
voice_config = {
    "name": "CEO Voice Clone",
    "description": "Custom voice for company announcements",
    "sample_files": ["ceo_sample1.mp3", "ceo_sample2.mp3"],
    "language": "en",
    "training_options": {
        "quality": "premium",
        "multilingual": True,
        "preserve_style": True
    }
}

result = create_and_monitor_custom_voice(voice_config)
```
