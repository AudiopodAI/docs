## Overview

AudioPod AI's Music Generation feature uses the advanced ACE-Step pipeline to transform text descriptions into original musical compositions. Our state-of-the-art system supports text-to-music, prompt-to-instrumental, and lyric-to-vocals generation, with full music editing capabilities including retake, repaint, extend, and edit operations.

### Key Features

- **ACE-Step Pipeline**: Advanced text-to-music generation using state-of-the-art AI models
- **Text-to-Music**: Generate complete musical compositions from natural language descriptions
- **Prompt-to-Instrumental**: Create instrumental tracks from text descriptions
- **Lyric-to-Vocals**: Generate vocal tracks from lyrics and text prompts
- **Music Editing**: Retake, repaint, extend, and edit existing musical tracks
- **Multiple Genres**: Support for 50+ music genres and styles
- **Customizable Length**: From 10 seconds to 10 minutes
- **High Quality**: Studio-grade audio output
- **Royalty-Free**: All generated music is royalty-free
- **Stem Separation**: Get individual instrument tracks
- **Real-time Generation**: Fast processing for quick iterations

## Quick Start

### Basic Music Generation

Generate original music by making a POST request to our ACE-Step pipeline. Your frontend application can call this endpoint directly.

```bash
curl -X POST "https://api.audiopod.ai/api/v1/music/text2music" \
  -H "Authorization: Bearer your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Upbeat pop song with electronic beats and catchy melody",
    "duration": 30,
    "genre": "pop",
    "mood": "energetic",
    "output_format": "mp3",
    "bpm": 128,
    "key": "C_major",
    "instrumental": false
  }' \
  --output generated_song.mp3
```

**Request Schema:**
```json
{
  "prompt": "string (required) - Text description of desired music",
  "duration": "number (required) - Duration in seconds (10-600)",
  "genre": "string (optional) - Music genre (pop, rock, jazz, classical, etc.)",
  "mood": "string (optional) - Mood/energy (energetic, calm, dark, uplifting)",
  "output_format": "string (optional) - mp3, wav, flac (default: mp3)",
  "bpm": "number (optional) - Beats per minute (60-200)",
  "key": "string (optional) - Musical key (C_major, A_minor, etc.)",
  "instrumental": "boolean (optional) - Generate instrumental only (default: false)",
  "style": "string (optional) - Additional style descriptors"
}
```

**Response Schema:**
```json
{
  "audio_data": "base64_encoded_audio_content",
  "title": "Energetic Pop Anthem",
  "duration": 30.5,
  "bpm": 128,
  "key": "C_major",
  "genre": "pop",
  "mood": "energetic",
  "format": "mp3",
  "size_bytes": 489726,
  "credits_used": 5,
  "stems_available": true,
  "stem_urls": {
    "vocals": "https://api.audiopod.ai/stems/vocals_abc123.mp3",
    "drums": "https://api.audiopod.ai/stems/drums_abc123.mp3",
    "bass": "https://api.audiopod.ai/stems/bass_abc123.mp3",
    "melody": "https://api.audiopod.ai/stems/melody_abc123.mp3"
  }
}
```

## Advanced Features

### Detailed Music Prompts

Create more specific music using detailed prompts:

```python
# Detailed prompt for specific musical elements
detailed_prompt = """
Create an uplifting acoustic folk song in the key of G major at 110 BPM.
Include acoustic guitar fingerpicking, subtle violin melody, and light percussion.
The mood should be hopeful and nostalgic, perfect for a sunrise scene.
Duration: 2 minutes with intro, verse, chorus, verse, chorus, bridge, outro structure.
"""

response = client.music_generation.create(
    prompt=detailed_prompt,
    duration=120,  # 2 minutes
    genre="folk",
    mood="hopeful",
    key="G major",
    bpm=110,
    structure="verse-chorus-verse-chorus-bridge-outro"
)
```

### Music with Stems

Get individual instrument tracks for mixing:

```python
response = client.music_generation.create(
    prompt="Jazz quartet with piano, bass, drums, and saxophone",
    duration=90,
    genre="jazz",
    include_stems=True,  # Return individual instrument tracks
    stem_format="wav"    # High quality for mixing
)

# Save individual stems
stems = response.stems
with open("jazz_piano.wav", "wb") as f:
    f.write(stems["piano"])
with open("jazz_bass.wav", "wb") as f:
    f.write(stems["bass"])
with open("jazz_drums.wav", "wb") as f:
    f.write(stems["drums"])
with open("jazz_saxophone.wav", "wb") as f:
    f.write(stems["saxophone"])
```

Transform existing musical ideas:

```python

)
```

```python
def create_video_background_music(video_info):

            "energy": "high"

            "prompt": "Calming meditation music with soft ambient tones and gentle progression",

            "mood": "peaceful",
            "energy": "low"
        },
        "cooking": {
            "prompt": "Upbeat acoustic music perfect for cooking shows with warm feeling",
            "genre": "acoustic",
            "mood": "cheerful",
            "energy": "medium"

    config = music_configs.get(video_info["type"], music_configs["corporate"])
```

### Podcast Intro/Outro Music

```python
def create_podcast_music(podcast_info):
    """Generate intro and outro music for podcasts"""

    # Create intro music
    intro_prompt = f"""
    Create a {podcast_info['mood']} podcast intro for '{podcast_info['title']}'.
    {podcast_info['genre']} style, {podcast_info['energy']} energy.
    Should build excitement and introduce the show professionally.
    Include space for voice-over announcement.
    """

    intro_music = client.music_generation.create(
        prompt=intro_prompt,
        duration=15,  # Short intro
        genre=podcast_info["genre"],
        mood=podcast_info["mood"],
        structure="intro-buildup-peak",
        fade_out=True,
        voice_space=True  # Leave space for voice-over
    )

    # Create outro music
    outro_prompt = f"""
    Create a {podcast_info['mood']} podcast outro that wraps up the show nicely.
    Should feel conclusive and leave listeners satisfied.
    Gentle fade out for smooth ending.
    """

    outro_music = client.music_generation.create(
        prompt=outro_prompt,

        genre=podcast_info["genre"],
        mood=podcast_info["mood"],
        structure="conclusion-fade",
        fade_in=True,
        fade_out=True
    )

    return {
        "intro": intro_music.audio_data,
        "outro": outro_music.audio_data
    }

# Example for a tech podcast
tech_podcast_music = create_podcast_music({
```

### AI-Generated Soundtracks

```python
def create_movie_soundtrack(movie_info, scenes):
    """Generate a complete movie soundtrack"""

    soundtrack = {}

    # Create main theme
    main_theme_prompt = f"""
    Create the main theme for {movie_info['title']}, a {movie_info['genre']} film.
    {movie_info['description']}
    Should be memorable and capture the essence of the story.
    """

    main_theme = client.music_generation.create(
        prompt=main_theme_prompt,
        duration=90,
        genre=movie_info["musical_style"],
        mood=movie_info["overall_mood"],
        structure="theme-development-climax-resolution",
        orchestral=True
    )

    soundtrack["main_theme"] = main_theme.audio_data

    # Create scene-specific music
    for scene_name, scene_info in scenes.items():
        scene_prompt = f"""
        Create music for the {scene_name} scene in {movie_info['title']}.
        Scene context: {scene_info['context']}
        Emotional arc: {scene_info['emotion_progression']}
        Should support the narrative and enhance the mood.
        """

        scene_music = client.music_generation.create(
            prompt=scene_prompt,
            duration=scene_info["duration"],
            genre=movie_info["musical_style"],
            mood=scene_info["primary_mood"],
            emotional_arc=scene_info["emotion_progression"],
            sync_to_action=True  # Music responds to scene pacing
        )

        soundtrack[scene_name] = scene_music.audio_data

    return soundtrack

# Example for an action thriller
thriller_soundtrack = create_movie_soundtrack(
```

## Pricing

Music generation pricing is based on duration and quality:

### Standard Pricing

- **100 credits per 30 seconds** of generated music
- Premium quality: 150 credits per 30 seconds
- Stem separation: +50 credits per track

### Cost Examples

| Duration         | Quality  | Credits | USD Cost |
| ---------------- | -------- | ------- | -------- |
| 30 seconds       | Standard | 100     | $0.10    |
| 1 minute         | Standard | 200     | $0.20    |
| 2 minutes        | Standard | 400     | $0.40    |
| 30 seconds       | Premium  | 150     | $0.15    |
| 1 minute + Stems | Standard | 350     | $0.35    |

### Cost Optimization

1. **Batch Generation**: Generate multiple variations in one request
2. **Appropriate Duration**: Generate only the length you need
3. **Standard Quality**: Use standard quality for testing and demos
4. **Reuse Elements**: Build on successful generations

## Best Practices

### Writing Effective Prompts

**✅ Good Prompts:**

- Be specific about genre, mood, and instruments
- Include tempo and energy level
- Describe the intended use case
- Mention specific musical elements you want

**❌ Avoid:**

- Vague descriptions like "good music"
- Conflicting style requests
- Overly complex prompts with too many requirements

### Examples of Effective Prompts

```python
effective_prompts = [
    # Good: Specific and clear
    "Upbeat indie rock song with jangly guitars, steady drums, and melodic bass line. Perfect for a summer road trip commercial. 120 BPM in E major.",

    # Good: Detailed atmosphere
    "Mystical ambient soundtrack for a fantasy video game forest level. Includes ethereal pads, subtle flute melody, and gentle percussion. Peaceful and magical mood.",

    # Good: Technical specifications
    "4-minute progressive rock instrumental with complex time signatures, guitar solos, and dynamic build-ups. Think Pink Floyd meets modern progressive metal.",

    # Avoid: Too vague
    "Make some good music",

    # Avoid: Conflicting styles
    "Heavy metal classical jazz fusion country ambient"
]
```
